Struktur om projectarbete.

1. Bygga upp en gemensam grund med header, footer, menu etc.
2. implimenta git
3. Överläggande design på sidan
4. Efter grunden är uppbyggd dela upp arbetsuppgifter dvs menu, contacts, etc.
5. gemensamt fixa quizen
6. Svara på frågor, fixa text information och bilder till sidan.
